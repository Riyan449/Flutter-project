## 📖 About CuRoCART

CuRoCART is a smart shopping cart mobile application developed using Flutter to modernize the retail shopping experience. The app connects users with an intelligent trolley system using Bluetooth and QR code technology, enabling seamless interaction between a mobile device and a physical shopping cart.

The project integrates mobile app development with IoT concepts to provide a digital shopping solution. Users can create accounts, log in securely, pair with a smart trolley, and connect to hardware devices for an enhanced and automated shopping experience.

CuRoCART demonstrates the practical use of Flutter for cross-platform development and Firebase for backend services, while also showcasing integration with Bluetooth communication and QR-based device pairing.

- 🔐 User Authentication (Login & Registration)

- ☁ Firebase Backend Support
- ⚡ Splash Screen Initialization
- 📱 Cross-platform Mobile Interface
- 🎨 Modern Material 3 UI Design
- 🔄 Named Route Navigation
- 🛠 Scalable Project Architecture

## 🎯 Project Objective

The main objective of CuRoCART is to provide a smart retail solution where users can interact with intelligent shopping trolleys using their smartphones. This application reduces manual shopping processes and introduces automation into retail environments through IoT integration.

## ⚙ Installation

### Clone the repository

```bash
git clone https://github.com/Riyan449/Flutter-project.git
```

### Navigate to project directory

```bash
cd Flutter-project
```

### Install dependencies

```bash
flutter pub get
```

### Run the app

```bash
flutter run
```

## 🔧 Firebase Setup

This project uses Firebase. To run successfully:

1. Create a Firebase project
2. Enable Authentication
3. Download configuration files:
   - `google-services.json` for Android
   - `GoogleService-Info.plist` for iOS
4. Add Firebase configuration
5. Run the project
